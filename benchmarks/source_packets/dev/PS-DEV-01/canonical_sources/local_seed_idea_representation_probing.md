# Local Seed Idea: Representation Probing

Design a small research proposal that tests whether probing results reveal stable structure in model representations or mostly reflect probe capacity, dataset artifacts, and analysis choices.

Constraints:

- The proposal should be feasible as a small research project.
- It should use the supplied representation-probing sources only.
- It should include falsifiable hypotheses, controls, baselines, and alternative explanations.
- It should distinguish correlational probing evidence from causal claims about model computation.
